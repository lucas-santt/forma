# Forma

Open-Source Project for tracking and organizing references for any creative project.

Collect and manage your inspirations and content that is important to you. Image, Text, Articles, Etc.

Inspired by [cosmos.so][cosmos] and [are.na][arena]

## Up-Coming Features

- User Login
- Text Archiving
- Articles Archiving
- Images Archiving
- Archives Linking
- Archive Sharing

[cosmos]: https://www.cosmos.so/
[arena]: https://www.are.na/

Made for the course MAC305 at Universidade de São Paulo (USP) in the first semester of 2025.
